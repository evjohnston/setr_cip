PUBLIC USE 2008 SDR DATA FILES
 
This CD-ROM contains data and related documentation for the public
use 2008 SDR data files.
 
The data is taken from the October 24, 2011 version of SESTAT maintained at NSF. 

DESCRIPTION OF FILES
--------------------

  Data and Programs
  -----------------
  
  The microdata are provided in various formats to facilitate their use with many software packages.
  The principal data format that is supported by NSF is SAS.  Technical support by NSF is limited to 
  the data files and the SAS programs listed here. NSF does not provide support to users of SPSS or 
  other packages, as SPSS and these other packages are not available at the agency.
  
  EPSD08.XPT - 2008 SDR data in SAS XPORT format
  EPSD08.DAT - 2008 SDR data in ASCII format
  EPSD08.sas7bdat - 2008 SDR data in SAS for Windows format
   
  FPSD08.SAS - a text file containing SAS FORMAT assignment statements
  LPSD08.SAS - a text file containing SAS LABEL assignment statements
  PPSD08.SAS - a text file containing SAS PROC FORMAT statements
  
  LdPSD08.SPS - a text file containing SPSS syntax for loading EPSD08.DAT (ASCII data) into SPSS
  NOTE: Technical support for SPSS users is not provided by NSF. The validity of this file 
  has not been verified by NSF, as SPSS is not available at the agency.
   
  LAYOUTPSD08.TXT - Input Layout for EPSD08.DAT
   
  Documentation and Related Files
  -------------------------------
  SDR08_PublicDataUser.PDF   - Survey instrument, 2008 SDR 
  
  PPSD08.PDF  - Summary variable codebook in Adobe PDF format.
  (Refer to http://www.nsf.gov/statistics/sestat/variableinfo.cfm for complete variable 
   documentation or if you do not have a copy of the Adobe PDF Reader).
   
  DPSD08.XLS - Data Dictionary Source/SAS name cross-walk file (EXCEL97 spreadsheet format)
   
   
  README.TXT - this read me file
