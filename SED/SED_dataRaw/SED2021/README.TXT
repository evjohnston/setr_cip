PUBLIC USE 2021 SURVEY OF DOCTORATE RECIPIENTS DATA FILES

This download file contains data and related documentation for the public
use 2021 Survey of Doctorate Recipients (SDR) data files.

The data included in this download file is taken from the January 09, 2023 version of the SDR data maintained at NSF. 

DESCRIPTION OF FILES
--------------------

  Data and Programs
  -----------------

  The microdata are provided in various formats to facilitate their use with many software packages.
  The principal data format that is supported by NSF is SAS.  Technical support by NSF is limited to 
  the data files and the SAS programs listed here. NSF does not provide support to users of SPSS or 
  other packages, as SPSS and these other packages are not available at the agency.

  EPSD21.DAT - 2021 SDR data in ASCII format
  epsd21.sas7bdat - 2021 SDR data in SAS for Windows format

  Fpsd21.sas - a text file containing SAS FORMAT assignment statements
  Lpsd21.sas - a text file containing SAS LABEL assignment statements
  Ppsd21.sas - a text file containing SAS PROC FORMAT statements

  Ldpsd21.sps - a text file containing SPSS syntax for loading EPSD21.DAT (ASCII data) into SPSS
     NOTE: Technical support for SPSS users is not provided by NSF. The validity of this file 
           has not been verified by NSF, as SPSS is not available at the agency.

  LAYOUTPSD21.TXT - Input Layout for EPSD21.DAT

  Documentation and Related Files
  -------------------------------
  https://ncses.nsf.gov/pubs/nsf23319#technical-notes - Technical notes

  psd21_Annotated.pdf - Survey instrument, 2021 SDR

  Ppsd21.html - Summary variable codebook in html, 508-compliant format.
  Ppsd21.pdf - Summary variable codebook in Adobe PDF format.

  Dpsd21.xlsx - Data Dictionary Source/SAS name crosswalk file (EXCEL spreadsheet format)

  README.TXT - this read me file
