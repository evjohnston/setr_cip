PUBLIC USE 2017 SURVEY OF DOCTORATE RECIPIENTS DATA FILES

This download file contains data and related documentation for the public
use 2017 Survey of Doctorate Recipients (SDR) data files.

The data included in this download file is taken from the March 06, 2019 version of the SDR data maintained at NSF. 

DESCRIPTION OF FILES
--------------------

  Data and Programs
  -----------------

  The microdata are provided in various formats to facilitate their use with many software packages.
  The principal data format that is supported by NSF is SAS.  Technical support by NSF is limited to 
  the data files and the SAS programs listed here. NSF does not provide support to users of SPSS or 
  other packages, as SPSS and these other packages are not available at the agency.

  EPSD17.DAT - 2017 SDR data in ASCII format
  epsd17.sas7bdat - 2017 SDR data in SAS for Windows format

  Fpsd17.sas - a text file containing SAS FORMAT assignment statements
  Lpsd17.sas - a text file containing SAS LABEL assignment statements
  Ppsd17.sas - a text file containing SAS PROC FORMAT statements

  Ldpsd17.sps - a text file containing SPSS syntax for loading EPSD17.DAT (ASCII data) into SPSS
  NOTE: Technical support for SPSS users is not provided by NSF. The validity of this file 
  has not been verified by NSF, as SPSS is not available at the agency.

  LAYOUTPSD17.TXT - Input Layout for EPSD17.DAT

  Documentation and Related Files
  -------------------------------
  SDR_SD_2017.pdf - Survey description

  psd17_Annotated.pdf - Survey instrument, 2017 SDR

  Ppsd17.pdf - Summary variable codebook in Adobe PDF format.

  Ppsd17.html - Summary variable codebook in html accessible format.

  Dpsd17.xlsx - Data Dictionary Source/SAS name cross-walk file (EXCEL spreadsheet format)

  README.TXT - this read me file
