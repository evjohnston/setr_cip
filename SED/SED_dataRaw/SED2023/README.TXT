PUBLIC USE 2023 SURVEY OF DOCTORATE RECIPIENTS DATA FILES

This download file contains data and related documentation for the public
use 2023 Survey of Doctorate Recipients (SDR) data files.

The data included in this download file is taken from the March 21, 2025 version of the SDR data maintained at NSF. 

DESCRIPTION OF FILES
--------------------

  Data and Programs
  -----------------

  The microdata are provided in various formats to facilitate their use with many software packages.
  The principal data format that is supported by NSF is SAS.  Technical support by NSF is limited to 
  the data files and the SAS programs listed here. NSF does not provide support to users of SPSS or 
  other packages, as SPSS and these other packages are not available at the agency.

  epsd23.csv - 2023 SDR data in comma-separated values format
  EPSD23.DAT - 2023 SDR data in ASCII format
  epsd23.sas7bdat - 2023 SDR data in SAS for Windows format

  Fpsd23.sas - a text file containing SAS FORMAT assignment statements
  Lpsd23.sas - a text file containing SAS LABEL assignment statements
  Ppsd23.sas - a text file containing SAS PROC FORMAT statements

  Ldpsd23.sps - a text file containing SPSS syntax for loading EPSD23.DAT (ASCII data) into SPSS
     NOTE: Technical support for SPSS users is not provided by NSF. The validity of this file 
           has not been verified by NSF, as SPSS is not available at the agency.

  LAYOUTPSD23.TXT - Input Layout for EPSD23.DAT

  Documentation and Related Files
  -------------------------------
  https://ncses.nsf.gov/surveys/doctorate-recipients/2023#technical-notes - Technical notes

  psd23_Annotated.pdf - Survey instrument, 2023 SDR

  Ppsd23.html - Summary variable codebook in html, 508-compliant format.
  Ppsd23.pdf - Summary variable codebook in Adobe PDF format.

  Dpsd23.xlsx - Data Dictionary Source/SAS name crosswalk file (EXCEL spreadsheet format)

  README.TXT - this read me file
