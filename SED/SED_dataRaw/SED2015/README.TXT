PUBLIC USE 2015 SESTAT (SURVEY OF DOCTORATE RECIPIENTS) DATA FILES

This download file contains data and related documentation for the public
use 2015 SDR data files.

The data is taken from the January 19, 2018 version of SESTAT maintained at NSF. 

DESCRIPTION OF FILES
--------------------

  Data and Programs
  -----------------

  The microdata are provided in various formats to facilitate their use with many software packages.
  The principal data format that is supported by NSF is SAS.  Technical support by NSF is limited to 
  the data files and the SAS programs listed here. NSF does not provide support to users of SPSS or 
  other packages, as SPSS and these other packages are not available at the agency.

  epsd15.xpt - 2015 SDR data in SAS XPORT format
  EPSD15.DAT - 2015 SDR data in ASCII format
  epsd15.sas7bdat - 2015 SDR data in SAS for Windows format

  Fpsd15.sas - a text file containing SAS FORMAT assignment statements
  Lpsd15.sas - a text file containing SAS LABEL assignment statements
  Ppsd15.sas - a text file containing SAS PROC FORMAT statements

  Ldpsd15.sps - a text file containing SPSS syntax for loading EPSD15.DAT (ASCII data) into SPSS
  NOTE: Technical support for SPSS users is not provided by NSF. The validity of this file 
  has not been verified by NSF, as SPSS is not available at the agency.

  LAYOUTPSD15.TXT - Input Layout for EPSD15.DAT

  Documentation and Related Files
  -------------------------------
  SDR_SD_2015.pdf - Survey description

  psd15_Annotated.pdf - Survey instrument, 2015 SDR

  Ppsd15.pdf  - Summary variable codebook in Adobe PDF format.

  Dpsd15.xlsx - Data Dictionary Source/SAS name cross-walk file (EXCEL spreadsheet format)

  2015_SDR_RecodeDocumentation.pdf - Recode Documentation for the 2015 SDR

  README.TXT - this read me file
